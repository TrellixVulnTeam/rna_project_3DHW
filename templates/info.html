Credits:

mirbase.org
We used the database files:
- hairpin.fa (Fasta format sequences of all miRNA hairpins),
- mature.fa (Fasta format sequences of all mature miRNA sequences)
as the basis of our database.



jsphylosvg.com
We used their open-source javascript library to render the graph reports
using the recent XML schema definition specified for phylogenetic trees, phyloXML.


phyloXML converter by Christian Zmasek
Java command line tool used to convert various phylogentic tree formats to phyloXML.
